# RotateImageView
The ImageView  can rotate in Android     旋转的ImageView 

使用：

private RotateImageView  rotateiv; 
rotateiv= new RotateImageView(this);  //初始化
	
rotateiv.stopRotate( boolean  isplay  );  //开始或者停止旋转
